# 複利計算機 
對，錢存進銀行會變多，可牛了
## STEP #1: 下載程式
阿你東西不弄到電腦上你是要怎麼用

要先安裝python環境
## STEP #2: 編輯setting
編輯 setting.xml的設定文件，要不然程式會不知道要幹嘛。設定文件看起來應該像這樣：
```
<data>
    <x>10000</x> <!-- 每年投資的金額 -->
    <s>12</s>    <!-- 年利率 -->
    <y>10</y>    <!-- 投資年份 -->
```
## STEP #3: 執行程式
就雙擊exe檔咩，還不簡單
## STEP #4: 看結果
結果會儲存在 result.txt，裡面能看到:

**本金**：你存的錢錢 

**年利率**：銀行每年給你幾趴 

**投資年分**：你存多久 

**總金額**：最後滾出的錢錢
